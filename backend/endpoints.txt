**Autenticação:**

*   `POST /login`: Para autenticar um usuário e obter um token de acesso.

**Empresas:**

*   `POST /empresas`: Criar uma nova empresa.
*   `GET /empresas`: Listar todas as empresas.
*   `GET /empresas/<id>`: Obter detalhes de uma empresa específica.
*   `PUT /empresas/<id>`: Atualizar os dados de uma empresa.

**Funcionários:**

*   `POST /funcionarios`: Cadastrar um novo funcionário.
*   `GET /funcionarios`: Listar todos os funcionários (provavelmente com filtros por empresa).
*   `GET /funcionarios/<id>`: Obter detalhes de um funcionário específico.
*   `PUT /funcionarios/<id>`: Atualizar os dados de um funcionário.
*   `DELETE /funcionarios/<id>`: Desligar um funcionário.
